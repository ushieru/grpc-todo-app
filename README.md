# gRPC - Todo app
    - gRPC
    - Typescript

## Instrucciones

> Aunque no es necesario puede agregar las variables de entorno en los .env
> uno en ./ y el otro en ./src/client/

```sh
sh build-all.sh
```

#### Start server

```sh
yarn start:server
```

#### Start client

```sh
yarn start:api
```

### Start react app
```sh
cd build/react
python -m http.server 3000
```
